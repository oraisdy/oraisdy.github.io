React Native动画小栗子

发表于 2017-06-27

今天学习了React Native中的Animated库，顺便用它给app做了一个登录界面的小动画。

这个动画其实是背景天空做横向运动与一张环形的icon图做旋转运动的并行执行。其实也是懒得换算位置所以直接把四个图标拼在了一张图里然后添加了一个圆形的遮罩层。😑

做一个小栗子的步骤分解：

  1. 使用Animated组件，如Animated.View，Animated.Image Animated.Text

  2. 使用Animated.Value设定一个或多个初始化值（其实就是你想要在动画中改变的属性的初始值）。

  3. 将动画中要改变的值绑定到动画目标的属性上。

    而当value为0.5时，会输出-180deg。也就是说对区间进行了映射。

  4. 通过Animated.timing等函数设定动画参数。

  5. 调用start启动。

介绍一下用到的几个重要的动画执行控制API：

  • Animated.sequence() 依次一个个的执行一系列动画
  • Animated.delay() 根据给定的一个延迟时间延迟启动接下来的动画
  • Animated.parallel() 并行执行两个或多个动画
  • Animated.stagger() 接受一个延迟时间和一系列动画数组，每隔延迟时间执行下一个动画（其实是插入了delay的parallel）
